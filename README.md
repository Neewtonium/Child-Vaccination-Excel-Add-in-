# Child-Vaccination-Excel-Add-in-
This is an Excel Add-in built using VBA; Tracks immunization schedules, highlights due vaccines, and monitors delays in Excel.





# Child Vaccine Tracker

A simple Excel Add-in that turns your vaccination data into clear, actionable insight.


---

## What you get:

✔ Know exactly which vaccines are due today
✔ See the next vaccines instantly
✔ Calculate child’s age in weeks and months automatically
✔ Detect delays in vaccination in seconds

No guessing. No manual calculations.


---

## How it works:

Enter Date of Birth →
Use simple formulas →
Excel does the thinking

```=GetDueVaccine(A2)```
```=GetNextVaccine(A2)```
```=GetChildAgeInWeeks(A2)```
```=GetVaccineGapInWeeks(A2,"PENTA1",B2)```

That’s it.
